# Section 5 — Scope and Definition of the Statistical Unit: Specific Instructions

You are generating **Section 5 (Scope of the Census and Definition of the Statistical Unit)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

scope, statistical unit, holding, agricultural holding, definition, crops, livestock, forestry, aquaculture, threshold, household, non-household, parcel, community, inclusion, exclusion, minimum size

## What to cover

### Scope of the census

Describe the types of agricultural activities covered by the census. Address whether the following were in scope (based only on what the source documents state):

- Crop production
- Livestock and poultry
- Forestry
- Aquaculture / fisheries
- Any other agricultural activities

### Statistical unit

Provide the official definition of the agricultural holding (or equivalent statistical unit) as stated in the source documents, including:

1. **Definition** — the official text or a faithful paraphrase of the definition used.
2. **Minimum inclusion thresholds** — any minimum size thresholds (area, head of livestock, etc.) below which a holding was not counted.
3. **Non-contiguous parcels** — how parcels not physically connected were treated (e.g. included if under the same management).
4. **Household and non-household sectors** — whether both household-sector and non-household-sector holdings were covered.
5. **Community-level data** — state separately whether community-level or communal data were collected.

## Writing rules for this section

- Use the official definition verbatim or as a faithful paraphrase — do not rephrase or simplify.
- State thresholds in the units documented (hectares, head, etc.).
- If any element is not documented, write exactly: *"Information on this point was not available in the source documents provided."*
- Do not include information about geographic coverage or exclusions — those belong in Section 6.
- Write in formal, neutral, third-person prose.
