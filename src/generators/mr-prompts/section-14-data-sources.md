# Section 14 — Data Sources: Specific Instructions

You are generating **Section 14 (Data Sources)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

source, document, report, publication, reference, bibliography, citation, authoring institution, title, year, place, URL, main report, technical report, questionnaire, methodology report

## What to cover

List all documents used to produce this Metadata Review. For each document, provide:

1. **Authoring institution** — the full official name of the organisation that produced the document.
2. **Full title** — the complete title of the document as it appears on the document itself.
3. **Year of publication** — the year the document was published or released.
4. **Place of publication** — the city or country of publication where documented.
5. **URL** — the URL from which the document was accessed, where available.

## Format requirement

Use a single consistent citation format for all entries. The recommended format is:

> Authoring Institution. *Full Title*. Place of Publication: Year. URL (if available).

Example:
> National Statistics Office (NSO), Nepal. *National Sample Census of Agriculture 2021/2022 — Main Report*. Kathmandu: 2022. https://example.gov.np/census

List only documents that were actually provided as source material for this Metadata Review. Do not add references to documents not included in the evidence.

## Writing rules for this section

- List each source document as a separate entry.
- Use the document's official title exactly as it appears on the document.
- Do not include references to FAO guidelines or external standards unless those documents were explicitly provided as source material.
- If publication details are not available for a document, include what is known and omit the rest.
- If information on this point was not available in the source documents provided, write exactly: *"Information on this point was not available in the source documents provided."*
- Write in formal, neutral, third-person prose.
