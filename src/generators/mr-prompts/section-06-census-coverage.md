# Section 6 — Census Coverage: Specific Instructions

You are generating **Section 6 (Census Coverage)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

coverage, geographic, territory, exclusion, excluded, area, province, district, region, threshold, size, cut-off, holding, sub-threshold, national

## What to cover

### Geographic coverage

State whether the census covered the **entire national territory**.

- If the census was nationally comprehensive, state this explicitly.
- If any geographic areas were excluded (e.g. conflict zones, remote areas, specific provinces), list them and state the reasons given in the source documents.

### Cut-off threshold and other exclusions

Describe any minimum holding-size thresholds below which holdings were not enumerated:

1. **Threshold value** — state the threshold in the units documented (e.g. hectares of agricultural land, number of livestock head).
2. **Excluded categories** — list any categories of holdings that were systematically excluded for reasons other than size (e.g. institutional farms, commercial estates, specific commodity producers).
3. **Sub-threshold treatment** — if sub-threshold holdings were recorded separately or counted but not fully enumerated, explain how this was handled.

## Writing rules for this section

- Distinguish clearly between geographic exclusions and size-based exclusions.
- State thresholds in the exact units documented.
- Only mention exclusions explicitly documented in the source; do not infer.
- If no exclusions existed, state this explicitly (e.g. "The census covered the entire national territory without geographic exclusions").
- If information is not available, write exactly: *"Information on this point was not available in the source documents provided."*
- Write in formal, neutral, third-person prose.

## Scope

This section covers geographic reach and cut-off rules only. Do not describe the definition of the statistical unit — that belongs in Section 5.
