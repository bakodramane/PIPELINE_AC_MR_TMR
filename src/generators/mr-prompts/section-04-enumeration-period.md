# Section 4 — Enumeration Period: Specific Instructions

You are generating **Section 4 (Enumeration Period)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

enumeration, fieldwork, data collection, period, start, end, phase, interview, enumerator, listing, pilot, schedule, duration

## What to cover

State the start and end dates of census fieldwork (the period during which enumerators visited holdings and collected data).

If fieldwork was conducted in phases (e.g. a listing phase followed by an interview phase, or separate phases for different regions or holding types), describe **each phase** separately:

1. **Phase name or identifier** — e.g. "listing phase", "main enumeration phase".
2. **Start and end dates** — state both dates in full (e.g. *1 March 2022* to *30 June 2022*).
3. **Holdings or areas covered** — describe which type of holdings or geographic areas each phase covered, where documented.

## Writing rules for this section

- State all dates in full (e.g. *15 November 2021*), not as month-year alone.
- If fieldwork was phased, describe each phase clearly and separately.
- Do not conflate the enumeration period with the reference date or reference period — those belong in Section 3.
- Do not include information about data processing or post-collection activities — those belong in Section 9.
- If the enumeration dates are not documented, write exactly: *"Information on this point was not available in the source documents provided."*
- Write in formal, neutral, third-person prose.
