# Section 8 — Use of Technology: Specific Instructions

You are generating **Section 8 (Use of Technology)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

technology, device, tablet, smartphone, mobile, software, application, CAPI, GPS, digital, platform, transmission, server, monitoring, dashboard, ODK, KoBoToolbox, CSPro, GIS, mapping, register, administrative

## What to cover

Describe the technological tools used during the census. Address all of the following where documented:

1. **Data collection devices and applications** — specify the devices used by enumerators (e.g. tablets, smartphones, handheld GPS units) and the software or application installed on them (e.g. ODK Collect, KoBoToolbox, CSPro, a custom application). Include the application name and version if documented.

2. **Field navigation** — describe any tools used for enumerator navigation (e.g. GPS receivers, digital maps, satellite imagery).

3. **Questionnaire management and data transmission** — describe how completed questionnaires were transmitted from field to central level (e.g. real-time mobile upload, batch transfer at the end of each day, physical submission).

4. **Use of administrative registers** — state whether administrative registers (e.g. agricultural holding registers, tax records, land registries) were used during the census, and if so, for what purpose (frame construction, coverage, post-collection matching).

5. **Data management, processing, and monitoring platforms** — describe any centralised software systems or dashboards used for data management, fieldwork monitoring, or quality control.

## Writing rules for this section

- Name software applications and platforms using their official names.
- Include version numbers only when explicitly documented.
- Do not describe questionnaire content or sampling design — those belong in Section 7.
- Do not describe data processing steps — those belong in Section 9.
- If technology use is not documented, write exactly: *"Information on this point was not available in the source documents provided."*
- Write in formal, neutral, third-person prose.
