# Section 3 — Reference Date and Period: Specific Instructions

You are generating **Section 3 (Reference Date and Period)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

reference date, reference period, reference day, agricultural year, stock, flow, calendar, interview, crop year, livestock, growing season

## What to cover

### Reference day(s)

State the calendar date(s) to which **stock variables** refer (e.g. number of livestock, area under permanent crops, land use on a specific day).

- If a single reference date was used for all stock variables, state it.
- If different reference dates were used for different variables or commodities, list each date and the variables it applies to.

### Reference period(s)

State the agricultural year or date range to which **flow variables** refer (e.g. crop production, area harvested during the season).

- If a single agricultural year was used, state its start and end dates.
- If multiple reference periods were used for different variables, list each period and the variables it applies to.

## Writing rules for this section

- State reference dates in full (e.g. *15 July 2022*), not as day-month alone.
- State reference periods with both start and end dates where documented.
- Distinguish clearly between stock variables (point in time) and flow variables (period of time).
- Do not infer dates that are not explicitly stated in the source documents.
- If reference dates or periods are not documented, write exactly: *"Information on this point was not available in the source documents provided."*
- Write in formal, neutral, third-person prose.

## Scope

This section covers only the reference date(s) and period(s). Do not describe the fieldwork schedule — that belongs in Section 4.
