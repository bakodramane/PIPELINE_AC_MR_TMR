# Section 1 — Historical Outline: Specific Instructions

You are generating **Section 1 (Historical Outline)** of an Agricultural Census Metadata Review.

## What to cover

Produce a concise, factual account of the history of agricultural censuses in the country. You must address all of the following points using only the evidence pages provided:

1. **Year of the first agricultural census** — state the calendar year in which the first census of agriculture was conducted in the country.

2. **Total number of censuses** — state the total number of agricultural censuses conducted up to and including the current operation.

3. **Years and types of each census** — for each census in the historical series, state the year and the type (e.g. general agricultural census, livestock census, complete enumeration, sample census, modular census). If the type is not explicitly stated in the source documents, state only the year.

4. **Current census number** — state explicitly which census number (e.g. first, second, third) the current operation represents in the series.

5. **Anomalous censuses** — note factually any census in the series that was incomplete, not published, suspended, or otherwise anomalous. State only what is documented; do not infer reasons.

## Writing rules for this section

- Write in formal, neutral, third-person prose.
- Do not use evaluative language ("successful", "comprehensive", etc.).
- If specific years or census types are not documented in the evidence, write: *"Information on this point was not available in the source documents provided."*
- Do not mention censuses that are not explicitly described in the evidence pages.
- Keep the section concise and proportionate to the information available — do not pad.
- Dates should be written in full where given (e.g. *2021/2022*); use numerals for quantities.

## Scope

This section covers only the historical series of agricultural censuses. Do not include information about the legal basis, methodology, scope, or results of the census — those belong in later sections.
