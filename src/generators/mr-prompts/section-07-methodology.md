# Section 7 — Methodology: Specific Instructions

You are generating **Section 7 (Methodology)** of an Agricultural Census Metadata Review.

## Keywords for evidence retrieval

methodology, method, modular, classical, register, frame, sample, enumeration, complete, questionnaire, interview, CAPI, paper, digital, design, stratification, cluster, probability, WCA 2020, essential items, thematic

## What to cover

This section requires a comprehensive description of the census methodology. Address **all** of the following sub-topics in order:

### Methodological modality

State the modality used: classical (traditional), modular, register-based, or integrated census/survey. If a combination was used, describe it.

### Frame construction

Describe how the census frame (the list of holdings or enumeration areas to be covered) was constructed:
- Source of the frame (e.g. population census listing, administrative registers, satellite imagery)
- Year of the frame and any updates made before the agricultural census

### Complete and/or sample enumeration

State whether a complete enumeration, a sample enumeration, or a combination was used:
- If sample: state which holdings or strata were fully enumerated and which were sampled.

### Sample design

If a sample was used, describe:
- Sample type (e.g. stratified, two-stage, cluster, probability proportional to size)
- Stratification variables and number of strata
- Sample size (number of enumeration areas or holdings selected)
- If no sample was used, state explicitly: "A complete enumeration was conducted; sample design is not applicable."

### Data collection method(s)

Describe how data were collected (e.g. direct interview, self-enumeration, CAPI, paper questionnaire). If multiple methods were used, describe each.

### Questionnaire(s) and items covered

- Describe the questionnaire structure and main thematic areas (e.g. land use, crops, livestock, farm machinery, labour).
- State the mode of data capture (paper or digital/CAPI).
- Note coverage of WCA 2020 essential items where explicitly documented in the source.

## Writing rules for this section

- Address every sub-topic listed above; use sub-headings as indicated.
- If a sub-topic is not documented, write exactly: *"Information on this point was not available in the source documents provided."*
- Do not include information about technology tools (devices, software) — those belong in Section 8.
- Write in formal, neutral, third-person prose.
